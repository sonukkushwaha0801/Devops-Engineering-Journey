Install and Setup Ansible for Implementing Policy as Code on AWS

1. Install boto3: ```pip install boto3```
2. Install AWS Collection: ```ansible-galaxy collection install amazon.aws```

Setup Vault
1. Create a password for vault: ```openssl rand -base64 2048 > vault.pass```
2. Add your AWS credentials using the below vault command: ```ansible-vault create group_vars/all/pass.yml --vault-password-file vault.pass```
