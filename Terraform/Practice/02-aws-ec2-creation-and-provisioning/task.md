# Task: AWS EC2 Provisioning using Terraform

# Objective

Provision an AWS EC2 instance using Terraform AWS Provider.

The task demonstrates:
- AWS provider configuration
- EC2 provisioning
- input variables
- tfvars usage
- outputs
- Terraform workflow
- AWS authentication

---

# Infrastructure Goal

Terraform should:
1. Authenticate with AWS
2. Initialize AWS provider
3. Create EC2 instance
4. Apply standard tags
5. Display instance details using outputs

---

# AWS Services Used

| Service | Purpose |
|---|---|
| EC2 | Virtual Machine |

---

# Concepts Covered

- AWS Provider
- EC2 Resource
- Variables
- terraform.tfvars
- Outputs
- Terraform Workflow
- AWS CLI Authentication

---

# Expected Result

After successful execution:
- EC2 instance should launch
- Terraform state should update
- Public IP should display
- Instance should appear in AWS Console
