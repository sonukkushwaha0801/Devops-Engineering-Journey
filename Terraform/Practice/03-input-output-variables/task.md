# Task: Terraform Input and Output Variables

# Objective

Provision an AWS EC2 instance using:
- input variables
- terraform.tfvars
- output variables

The task demonstrates reusable and configurable Terraform infrastructure.

---

# Infrastructure Goal

Terraform should:
1. Accept user-defined variables
2. Read values from terraform.tfvars
3. Create EC2 instance dynamically
4. Display infrastructure outputs

---

# Concepts Covered

- Input Variables
- Variable Types
- terraform.tfvars
- Output Variables
- Variable References
- Dynamic Infrastructure Configuration

---

# AWS Services Used

| Service | Purpose         |
| ------- | --------------- |
| EC2     | Virtual Machine |

---

# Expected Result

After successful execution:
- EC2 instance should launch
- Variables should control infrastructure values
- Outputs should display infrastructure details
