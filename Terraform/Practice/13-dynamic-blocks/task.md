# Task: Terraform Dynamic Blocks

# Objective

Provision AWS Security Groups using:
- dynamic blocks
- variable-driven ingress rules
- reusable Terraform logic

The task demonstrates scalable Terraform configuration generation.

---

# Infrastructure Goal

Terraform should:
1. Create Security Group dynamically
2. Generate ingress rules using dynamic blocks
3. Reuse variable-driven configuration
4. Reduce repetitive Terraform code

---

# Concepts Covered

- dynamic blocks
- reusable ingress rules
- iterative configuration
- scalable Terraform logic
- production Terraform patterns

---

| Service        | Purpose         |

# Expected Result

After successful execution:
- multiple ingress rules should generate dynamically
- Terraform configuration should remain reusable and scalable
- infrastructure should avoid repetitive code blocks
