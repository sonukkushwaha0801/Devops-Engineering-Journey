# Steps to Execute the Task

# Step 1: Install Terraform

## Linux Installation

Download Terraform:
```bash
wget https://releases.hashicorp.com/terraform/1.8.5/terraform_1.8.5_linux_amd64.zip
```

Install unzip:
```bash
sudo apt update

sudo apt install unzip -y
```

Unzip Terraform:
```bash
unzip terraform_1.8.5_linux_amd64.zip
```

Move binary:
```bash
sudo mv terraform /usr/local/bin/
```

Verify installation:
```bash
terraform version
```

Expected Output:
```text
Terraform v1.8.5
```

# Step 4: Initialize Terraform

Command:
```bash
terraform init
```

Purpose:
- download local provider
- initialize Terraform working directory
- create `.terraform/`

Expected:
```text
Terraform has been successfully initialized
```

---

# Common Errors

## terraform: command not found

Reason:
Terraform not installed correctly.

Fix:
Verify:
```bash
terraform version
```

---

## Failed to query provider packages

Reason:
No internet connectivity.

Fix:
Ensure internet access available.

---

## Permission denied

Reason:
Insufficient file permissions.

Fix:
Use:
```bash
sudo
```

where required.
