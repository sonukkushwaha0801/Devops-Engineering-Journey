# Task: Terraform Modules Practical

# Objective

Provision AWS EC2 infrastructure using reusable Terraform modules.

The task demonstrates:
- module creation
- module usage
- reusable infrastructure
- variable passing
- module outputs

---

# Infrastructure Goal

Terraform should:
1. Create reusable EC2 module
2. Pass variables from root module
3. Provision EC2 instance using module
4. Expose outputs from module

---

# Concepts Covered

- Terraform Modules
- Child Modules
- Root Module
- Module Inputs
- Module Outputs
- Reusable Infrastructure

---

# AWS Services Used

| Service | Purpose         |
| ------- | --------------- |
| EC2     | Virtual Machine |

---

# Expected Result

After successful execution:
- EC2 instance should launch
- infrastructure should use reusable module architecture
- outputs should display module-generated values
